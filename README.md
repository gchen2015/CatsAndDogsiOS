# Cats And Dogs Talk: Swift iOS App
This is the Swift 3.0 (for iOS) version of the cats and dogs app example code. It connects to the [backend for frontend](https://github.com/RomainPiel/CatsAndDogs-kotlin-bff) to get it's view model data to render.
